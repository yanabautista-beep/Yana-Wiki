# KSA Riyadh A/B Test — CPP + Instant Experience

## Hypothesis

CPP + Instant Experience format would outperform the control Meta format for Home Cleaning in KSA.

## Mechanic

- **Channel:** Meta
- **Market:** KSA — Riyadh
- **Variant A (Control):** Existing campaign format
- **Variant B (Treatment):** CPP + Instant Experience
- **Voucher added:** MLK50

## Results

CPP + Instant Experience format performed better. Riyadh AB test paused, continued with winner. Format rolled out to Jeddah.

## Next Step

- Riyadh: Continued with CPP + Instant Experience
- Jeddah: Took old campaign live with CPP + Instant Experience + MLK50
- Scale: Expanded to both cities with the winning format

## Status

Resolved — winner applied to both Riyadh and Jeddah. Decision made 30.10.2025.
