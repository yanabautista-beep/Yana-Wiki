# App Marketing

## Current Status

App install campaigns running via Meta, TikTok, and Snapchat. UAE Home Cleaning is the primary app campaign. KSA App Install campaigns being set up for Dammam-Khobar. App sessions and conversion rates are monitored regularly. Attribution tracked via Adjust.

## Key Campaigns

- **UAE Home Cleaning App Install** — Meta + TikTok active
- **KSA Dammam App Install** — TikTok + Snapchat (pending creatives)

## Performance Observations

- Jan 2024 UAE Home Cleaning app spend: Meta $11,929, TikTok $10,053, Total $21,983 (10% = $2,198)
- App visitor sessions increased but conversion rate decreased (Sep 2024) — attributed to trend, not campaign issue
- Increasing app campaign budgets did not improve conversions — confirmed as organic trend decrease
- Conversion rate on app decreased even as web visitor sessions increased (Jan 2025)
